# 第 3 頁｜前處理：4 種資料，4 種方法

## PDF：圖片式掃描 → OCR 辨識

- **問題：** PDF 每頁都是圖片，無法直接讀文字
- **方法：** PyMuPDF 渲染 → Tesseract OCR（繁中＋簡中＋英）
- **結果：** 9 頁全部辨識，含案號、被告、犯罪事實、筆錄

## WAV：60 分鐘廣播 → 逐字稿

- **問題：** IMA ADPCM 格式、台語混雜
- **方法：** Whisper medium 模型轉錄，加時間戳記
- **結果：** 1,551 段，60 分鐘完整逐字稿

## XLS：複雜多層表頭 → 可查詢表格

- **問題：** 12 工作表、合併儲存格、欄位名稱全 Unnamed
- **方法：** xlrd 逐格讀取，手動重建縣市×性別×年齡結構
- **結果：** 25 縣市，每縣市男/女/計，完整年齡組別

## ZIP：100 份公文 → 文字索引

- **方法：** zipfile 解壓 → PyMuPDF 萃取 → 逐份 Markdown
- **結果：** 201 個輸出檔，含索引表

# 第 4 頁｜自驗 20 題：成功經驗與地雷

## ✅ 成功的部分

| 來源 | 驗證題例 | 結果 |
|------|---------|------|
| PDF | 被告王小明的帳戶如何遭利用？ | ✓ 精確引用第4頁犯罪事實 |

| WAV | 余勝福參選哪個選區？ | ✓ 大寮-林門市議員 |

## ⚠️ 踩到的地雷

| 地雷 | 說明 | 解法 |
|------|------|------|
| PDF 是圖片 | 直接讀文字得到 0 字 | 換 OCR 方案 |
| XLS 合併儲存格 | pandas 讀出全是 Unnamed | 改用 xlrd 逐格讀取 |
| WAV 格式 IMA ADPCM | Python wave 模組不支援 | 改用 soundfile |
| 模型下載封鎖 | 雲端環境封鎖 HuggingFace | 改到本地執行 Whisper |
| Prompt 太長 | 100 份公文全塞進去超 token | 改為依問題動態選 references |
