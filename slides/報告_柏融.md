# 第 3 頁｜前處理：4 種資料，4 種方法

## PDF：圖片式掃描 → OCR 辨識

- **問題：** PDF 每頁都是圖片，無法直接讀文字
- **方法：** PyMuPDF 渲染 → Tesseract OCR（繁中＋簡中＋英）
- **結果：** 9 頁全部辨識，含案號、被告、犯罪事實、筆錄

## WAV：60 分鐘廣播 → 逐字稿

- **問題：** IMA ADPCM 格式、台語混雜
- **方法：** Whisper medium 模型轉錄，加時間戳記
- **結果：** 1,551 段，60 分鐘完整逐字稿

## XLS：複雜多層表頭 → 可查詢表格

- **問題：** 12 工作表、合併儲存格、欄位名稱全 Unnamed
- **方法：** xlrd 逐格讀取，手動重建縣市×性別×年齡結構
- **結果：** 25 縣市，每縣市男/女/計，完整年齡組別

## ZIP：100 份公文 → 文字索引

- **方法：** zipfile 解壓 → PyMuPDF 萃取 → 逐份 Markdown
- **結果：** 201 個輸出檔，含索引表

## ⚠️ 前處理地雷

| 地雷 | 說明 | 解法 |
|------|------|------|
| PDF 是圖片 | 直接讀文字得到 0 字 | 換 OCR 方案（Tesseract） |
| XLS 合併儲存格 | pandas 讀出全是 Unnamed | 改用 xlrd 逐格讀取 |
| WAV 格式 IMA ADPCM | Python wave 模組不支援 | 改用 soundfile |
| 模型下載封鎖 | 雲端封鎖 HuggingFace | 改到本地執行 Whisper |

## 🚀 部署測試（Railway）發現的問題

| 問題 | 根本原因 | 解法 |
|------|---------|------|
| Build 失敗：無法偵測語言 | Railpack 掃根目錄找不到 `requirements.txt` | 在根目錄加 `requirements.txt` |
| Build 失敗：找不到 start command | `main.py` 不在根目錄 | 在根目錄加 `main.py` 引入 hermes-agent app |
| 向量搜尋未啟用（fulltext） | `chromadb` 未加入 `requirements.txt` | 補加 `chromadb==1.5.9` |
| ZIP 公文查不到特定編號 | TOP_K=5 不夠，100 份公文排名太後面 | ZIP 的 TOP_K 從 5 提高到 15 |
| 429 Too Many Requests | 連續請求觸發 NCHC API 流量限制 | 加 retry + exponential backoff（10s/20s/40s） |
| 中文答題準確率偏低 | `all-MiniLM-L6-v2` 是英文模型 | 換成 `paraphrase-multilingual-MiniLM-L12-v2` |
